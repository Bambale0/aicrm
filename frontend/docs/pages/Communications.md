# История коммуникаций (Communications)

## Обзор
Страница истории коммуникаций предоставляет унифицированный интерфейс для просмотра всех взаимодействий с клиентами через различные каналы связи.

## Функциональность

### 💬 Объединенная история
- **Все каналы в одном месте**: Telegram, Avito, Email, Phone
- **Хронологическая сортировка** (новые сверху)
- **Типы сообщений**: входящие/исходящие
- **Статусы доставки**

### 🔍 Фильтрация и поиск
- **Фильтр по каналам** (Telegram, Avito, Email, Phone)
- **Фильтр по направлению** (входящие/исходящие)
- **Поиск по тексту** сообщений
- **Фильтр по клиентам**
- **Временной диапазон**

### 📊 Статистика каналов
- **Количество сообщений** по каналам
- **Входящие vs исходящие**
- **Активность по времени**
- **Популярные каналы**

## API Endpoints

```typescript
GET /telegram/communications     // Коммуникации Telegram
GET /avito/communications       // Коммуникации Avito (плейсхолдер)
GET /email/communications       // Коммуникации Email (плейсхолдер)
GET /phone/communications       // Коммуникации Phone (плейсхолдер)
```

## Структура данных

```typescript
interface Communication {
  id: string;
  channel: 'telegram' | 'avito' | 'email' | 'phone';
  direction: 'inbound' | 'outbound';
  customer_id?: number;
  customer_name?: string;
  message: string;
  created_at: string;
  status?: string;
  metadata?: any;
}
```

## Интеграция с CRM

### 👥 Клиенты
```typescript
// Связь сообщений с клиентами
communication.customer_id → customer.id
communication.customer_name → customer.full_name
```

### 📦 Заказы
```typescript
// Коммуникации по заказам
order.communications[] → communications
```

### 🤖 AI интеграция
```typescript
// AI анализ тональности сообщений
communication.sentiment → ai.analysis
```
