# Управление Email (EmailManagement)

## Обзор
Страница управления Email предоставляет комплексный интерфейс для создания, управления и анализа email шаблонов и рассылок.

## Функциональность

### 📧 Шаблоны Email
- **Создание шаблонов** с переменными
- **Визуальный редактор** (плейсхолдер)
- **Тестирование шаблонов**
- **Управление версиями**

### 📊 Статистика рассылок
- **Общее количество** отправленных писем
- **Показатели доставки** (доставлено/открыто/клики)
- **Конверсионные метрики**
- **Временные тренды**

### 📋 Логи отправки
- **История всех отправок**
- **Статусы доставки** по каждому письму
- **Сообщения об ошибках**
- **Время доставки**

## API Endpoints

```typescript
GET    /email/templates         // Список шаблонов
POST   /email/templates         // Создание шаблона
PUT    /email/templates/{id}    // Обновление шаблона
DELETE /email/templates/{id}    // Удаление шаблона
POST   /email/test              // Тестирование email
GET    /email/status            // Статус отправки
GET    /email/service-status    // Статус сервиса
```

## Структура шаблонов

```typescript
interface EmailTemplate {
  id: string;
  name: string;
  subject: string;
  body: string;
  variables: string[];
  is_active: boolean;
  created_at: string;
  updated_at: string;
}
```

## Переменные в шаблонах

### 📝 Доступные переменные
```handlebars
{{customer_name}}     // Имя клиента
{{order_number}}      // Номер заказа
{{order_total}}       // Сумма заказа
{{delivery_date}}     // Дата доставки
{{support_email}}     // Email поддержки
{{company_name}}      // Название компании
```

### 🔄 Примеры использования
```html
Уважаемый {{customer_name}},

Ваш заказ №{{order_number}} на сумму {{order_total}}₽
готов к отправке. Ожидаемая дата доставки: {{delivery_date}}.

С уважением,
Команда {{company_name}}
```
