# Логи автоматизации (AutomationLogs)

## Обзор
Страница логов автоматизации предоставляет детальный мониторинг и анализ выполнения автоматических процессов в системе.

## Функциональность

### 📊 Статистика выполнений
- **Общее количество** выполнений
- **Успешные vs неудачные** выполнения
- **Процент успешности**
- **Среднее время** выполнения

### 📋 Детальные логи
- **Список всех выполнений** с детализацией
- **Статус выполнения** (success, error, running, pending)
- **Время начала и окончания**
- **Длительность выполнения**
- **Сообщения об ошибках**

### 🔍 Фильтрация и поиск
- **Фильтр по статусу** выполнения
- **Фильтр по типу сущности** (order, customer, task)
- **Фильтр по роботу**
- **Временной диапазон**
- **Поиск по тексту**

## API Endpoints

```typescript
GET /automation/analytics/executions  // Статистика и логи выполнений
GET /automation/analytics/robots      // Производительность роботов
GET /automation/analytics/actions     // Статистика действий
GET /automation/analytics/errors      // Анализ ошибок
GET /automation/analytics/processes   // Эффективность процессов
GET /automation/analytics/hourly      // Почасовая статистика
```

## Структура данных лога

```typescript
interface ExecutionLog {
  id: string;
  entity_type: string;      // order, customer, task
  entity_id: number;        // ID сущности
  robot_id?: number;        // ID робота
  robot_name?: string;      // Имя робота
  stage_id?: number;        // ID стадии
  stage_name?: string;      // Имя стадии
  action_type?: string;     // Тип действия
  status: 'success' | 'error' | 'running' | 'pending';
  started_at: string;       // Время начала
  completed_at?: string;    // Время окончания
  duration_ms?: number;     // Длительность в мс
  error_message?: string;   // Сообщение об ошибке
  metadata?: any;          // Дополнительные данные
}
```

## Статусы выполнений

### ✅ Success (Успешно)
- **Зеленый индикатор**
- **Завершенное выполнение** без ошибок
- **Все действия выполнены**

### ❌ Error (Ошибка)
- **Красный индикатор**
- **Критическая ошибка** в выполнении
- **Детальное сообщение** об ошибке
- **Возможность повторного запуска**

### 🔄 Running (Выполняется)
- **Синий индикатор** с анимацией
- **Активное выполнение**
- **Прогресс выполнения**

### ⏳ Pending (Ожидает)
- **Желтый индикатор**
- **Ожидание в очереди**
- **Подготовка к выполнению**

## Интеграция с другими компонентами

### 🤖 Роботы (Robots)
```typescript
// Логи связаны с конкретными роботами
log.robot_id → robot.id
log.robot_name → robot.name
```

### ⚙️ Стадии (Stages)
```typescript
// Логи привязаны к стадиям процессов
log.stage_id → stage.id
log.stage_name → stage.name
```

### 🎯 Триггеры (Triggers)
```typescript
// Триггеры создают логи выполнений
trigger.fired → log.created
```
