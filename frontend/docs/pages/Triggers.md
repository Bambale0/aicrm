# Триггеры автоматизации (Triggers)

## Обзор
Страница управления триггерами позволяет настраивать автоматические события, которые запускают бизнес-процессы при определенных условиях.

## Функциональность

### 🎯 Просмотр триггеров
- **Список активных триггеров** с условиями срабатывания
- **Статус активности** каждого триггера
- **Тип события** (created, updated, status_changed)
- **Целевая стадия** для перехода

### ➕ Создание триггера
- **Выбор типа сущности** (order, customer, task)
- **Настройка события** (создание, обновление, изменение статуса)
- **Установка условий** срабатывания
- **Выбор целевой стадии**

### ✏️ Редактирование триггера
- **Изменение условий** срабатывания
- **Переназначение стадии**
- **Обновление описания**
- **Включение/выключение** триггера

### 🗑️ Удаление триггера
- **Проверка зависимостей**
- **Безопасное удаление** с подтверждением

## API Endpoints

```typescript
GET    /automation/triggers/           // Список триггеров
POST   /automation/triggers/           // Создание триггера
GET    /automation/triggers/{id}       // Получение триггера
PUT    /automation/triggers/{id}       // Обновление триггера
DELETE /automation/triggers/{id}       // Удаление триггера
```

## Типы событий

### 📝 Создание сущности
```typescript
{
  entity_type: "order",
  event_type: "created",
  conditions: {},
  target_stage_id: 1
}
```

### 🔄 Обновление сущности
```typescript
{
  entity_type: "customer",
  event_type: "updated",
  conditions: {
    field: "status",
    operator: "equals",
    value: "active"
  },
  target_stage_id: 2
}
```

### 📊 Изменение статуса
```typescript
{
  entity_type: "task",
  event_type: "status_changed",
  conditions: {
    from_status: "pending",
    to_status: "in_progress"
  },
  target_stage_id: 3
}
```

## Конструктор условий

### Простые условия
- **Равенство**: `field == value`
- **Неравенство**: `field != value`
- **Больше/меньше**: `field > value`

### Сложные условия
- **И/ИЛИ логика** для комбинаций
- **Вложенные условия**
- **Временные ограничения**

### Примеры условий
```typescript
// Заказ на сумму > 10000
{
  field: "total_amount",
  operator: "greater_than",
  value: 10000
}

// Клиент из Москвы
{
  field: "city",
  operator: "equals",
  value: "Москва"
}

// Задача просрочена
{
  field: "due_date",
  operator: "less_than",
  value: "now"
}
```

## Интеграция с автоматизацией

### 🤖 Роботы
```typescript
// Триггеры запускают роботов
trigger.fired → robot.execute()
```

### ⚙️ Процессы
```typescript
// Триггеры инициируют процессы
trigger.fired → process.start()
```

### 📊 Логи
```typescript
// Все срабатывания логируются
{
  trigger_id: 1,
  entity_type: "order",
  entity_id: 123,
  fired_at: "2025-01-15T10:30:00Z"
}
```
